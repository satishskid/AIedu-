## Getting Started

### What is EduAI?

EduAI is an AI-powered programming education platform designed specifically for K-12 schools. It provides:

- **Interactive Programming Lessons** with AI tutoring
- **Gamified Learning Experience** with points, badges, and leaderboards
- **Comprehensive Teacher Tools** for classroom management
- **Offline Functionality** that works without internet
- **Progress Tracking** and detailed analytics

### System Requirements

- **Modern Web Browser**: Chrome, Firefox, Safari, or Edge (latest versions)
- **Internet Connection**: Required for initial setup and sync (optional for daily use)
- **Device**: Desktop, laptop, or tablet with keyboard
- **Screen Resolution**: Minimum 1024x768 pixels

### First Time Access

1. **Receive Login Credentials**: Your school administrator will provide:
   - Website URL (e.g., `https://your-school.eduai.com`)
   - Username and temporary password
   - Role assignment (Administrator, Teacher, or Student)

2. **Initial Login**:
   - Visit the provided URL
   - Enter your credentials
   - Change your password when prompted
   - Complete your profile setup

1. **Access License Dashboard**:
   - Navigate to `Settings > License Management`
   - View current license status and usage

2. **Activate School License**:
   ```
   1. Click "Activate New License"

   3. Verify school information
   4. Confirm activation
   ```

3. **Monitor License Usage**:

   - Go to `Users > Add New User`
   - Fill in required information:

   3. Upload completed CSV file
   4. Review and confirm import
   5. Send credentials to all new users

## Troubleshooting

### Common Issues and Solutions

#### Login Problems

**Issue**: Cannot log in to the system
**Solutions**:
1. Check username and password spelling
2. Ensure Caps Lock is off
3. Try password reset if available
4. Clear browser cache and cookies
5. Contact your teacher or administrator

**Issue**: Forgot password
**Solutions**:
1. Use "Forgot Password" link on login page
2. Check email for reset instructions
3. Contact administrator for manual reset

#### Technical Issues

**Issue**: Code editor not loading
**Solutions**:
1. Refresh the browser page
2. Check internet connection
3. Disable browser extensions temporarily
4. Try a different browser
5. Clear browser cache

**Issue**: Slow performance
**Solutions**:
1. Close unnecessary browser tabs
2. Check internet connection speed
3. Restart browser
4. Update browser to latest version
5. Contact technical support

#### Assignment Issues

**Issue**: Cannot submit assignment
**Solutions**:
1. Check if deadline has passed
2. Ensure all required fields are completed
3. Verify code runs without errors
4. Try submitting from different browser
5. Contact teacher immediately

**Issue**: Lost work/code disappeared
**Solutions**:
1. Check auto-save history
2. Look in browser's local storage
3. Check if logged into correct account
4. Contact teacher for backup recovery

### Browser Compatibility

#### Recommended Browsers
- **Chrome**: Version 90 or later
- **Firefox**: Version 88 or later
- **Safari**: Version 14 or later
- **Edge**: Version 90 or later

#### Browser Settings
1. **Enable JavaScript**: Required for all functionality
2. **Allow Cookies**: Needed for login and preferences
3. **Enable Local Storage**: Required for offline functionality
4. **Disable Ad Blockers**: May interfere with platform features
